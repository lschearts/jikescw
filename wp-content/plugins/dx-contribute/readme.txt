The shortcut Submission wordpress plugin. 快捷投稿插件

== Description ==

create a Submission page, allowing visitors through the page shortcut Contributor.

创建一个投稿页面，允许匿名用户在线快捷投稿。

详情请浏览：[http://www.daxiawp.com/dx-contribute.html](http://www.daxiawp.com/dx-contribute.html)

== Installation ==

可以通过以下两种方法的其中一种来安装DX-Contribute 插件：

1. 将下载的文件解压缩，然后将`dx-contribute`文件夹 上传到 `/wp-content/plugins/`目录，在插件后台启用

2. 直接在后台-安装插件，搜索'dx-contribute'，按照提示安装启用
